## Run Locally

Clone the project

```bash
  git clone git@github.com:TheDa21crj/Portfolio.git
```

Go to the project directory

```bash
  cd Portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
